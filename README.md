# Auto_Test_Checker
Телеграм бот, автоматически проверяющий тесты с изображений


# Возможности:
 - получить проверенный тест по изображению
 - настроить порог проверки одного поля по команде
 
Результат пока не обрабатывается - можно дописать, для полной автоматичской проверки

В примере бот настроен для некачественных изображений(
Например, полученных с камеры телефона)
